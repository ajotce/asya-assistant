# Asya Assistant (MVP)

Asya — персональный ИИ-ассистент в формате PWA-веб-чата с локальным backend в Docker.

Текущий статус: подготовлен базовый каркас проекта (Этап 0 по `asya-mvp-development-plan.md`), без реализации бизнес-логики.

## Что входит в MVP
- PWA-интерфейс (русский UI) для Mac/iPhone.
- Текстовый чат со streaming-ответами.
- Выбор модели через VseLLM (через backend).
- Временная сессия без долговременной памяти.
- Поддержка файлов (PDF, DOCX, XLSX, изображения) в рамках текущей сессии.

## Что не входит в MVP
- Todoist, календари, CRM, контакты.
- Веб-поиск.
- Голосовой ввод и озвучка.
- Долговременная память и история чатов.
- Регистрация и полноценная авторизация.
- Production/VPS деплой в рамках MVP.

## Быстрый старт (каркас)
1. Создать локальный `.env`:
```bash
cp .env.example .env
```

2. Базовый запуск контейнера backend (каркас):
```bash
docker compose up --build
```

## Полезные команды
```bash
make dev
make test
make lint
make build-frontend
```

## Правила Git в разработке
- Изменения вносятся маленькими этапами по плану MVP.
- После каждого завершенного шага выполняются проверки, затем commit и push в GitHub.
- Для безопасности предпочтительна отдельная рабочая ветка под этап.

## Структура
- `frontend/` — будущий React + Vite + TypeScript frontend.
- `backend/` — будущий FastAPI backend.
- `docs/` — документация проекта и архитектурные решения.
